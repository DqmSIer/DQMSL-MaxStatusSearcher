使い方の説明

まずは、コードをコンパイルして実行可能ファイルを作る
----------
$ make
g++ src/main.cpp -std=c++11 -O2
----------

以下のように実行する
----------

間違えた時に手入力し直すのは面倒なので、あらかじめ転生回数とステータスを
ファイルに書いておいてリダイレクトで入力すると楽です。

----------
# 
$ ./a.out < buon.txt
----------

入力データサンプルとして ulsl.txt/buon.txt を置いたのでこれを参考に書いてください。
ステータス値の入力時点で行頭がハイフン - になっていると無視されます。
これを利用して、どこからどこまでがどのモンスターのステータスか分かるように書いてます。
